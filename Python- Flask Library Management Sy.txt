Python- Flask Library Management System


This Flask application allows users to manage a library system. It includes features such as adding, searching, editing, and checking out books.


To run this project, follow these steps:
install flask into your py-charm application, install all required packages, such as sqlAlchemy, flask, flask login, models..... 

Access the application through the web browser at `http://localhost:5000`.
Features
- Add new books to the system.
- Search for books by title, author, or ISBN.
- Edit book details.
- Register new users and manage checkouts

Configure your database connection.
